專案說明

  • 網站主要使用 Vue 3 和 Vue Cli 架設而成
  • 引入 Vue Router 與 Vue Axios 等套件
  • 引入 Bootstrap 5 樣式
  • 開發後台管理系統
  • 加入 ESLint Airbnb 風格
  • 透過 mitt 傳遞跨元件資料
  • Layout 使用 Bootstrap 5 結合 Sass 切版
  • 使用 Material icons

前台

  • 電商展示：首頁、商品列表、商品介紹等
  • 文章展示：文章列表、單一文章頁，結合電商商品推薦
  • 購物車：新增、修改、刪除
  • 優惠券
  • 結帳

後台

  • 管理者登入
  • 商品管理
  • 優惠券管理
  • 訂單管理
  • 文章管理
